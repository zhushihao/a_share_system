# Quant Workbench 报告目录

```
reports/
├── selfcheck/          # 系统排查报告（每小时整点）
│   └── SYSTEM_LOOP_CHECK_REPORT_YYYY-MM-DD_HHMM.md
├── ux/                 # 用户体验报告（每小时整点）
│   └── UX_CHECK_REPORT_YYYY-MM-DD_HHMM.md
└── postmarket/         # 盘后报告（当前未使用）
```

## 文件命名

- 系统排查：`SYSTEM_LOOP_CHECK_REPORT_YYYY-MM-DD_HHMM.md`
- 用户体验：`UX_CHECK_REPORT_YYYY-MM-DD_HHMM.md`

## 查看方式

在 VSCode 中打开 `a_share_system` 文件夹，浏览 `reports/` 目录。

按修改时间排序，取最新的文件阅读。
